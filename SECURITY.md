# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Visible Cursor, please report it responsibly.

To report a vulnerability, please email [security@notuntoward.com](mailto:security@notuntoward.com) with details of the issue. Do not open a public issue for security vulnerabilities.

I will acknowledge your report within 48 hours and provide a timeline for a fix.

## Supported Versions

Only the latest release version receives security updates. Please update to the latest version if you discover a vulnerability in your current version.
